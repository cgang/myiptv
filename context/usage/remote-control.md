# Remote Control Navigation

Designed for Android TV remote controllers.

## Button Mapping

| Button | Action |
|--------|--------|
| **OK** | Open playlist / Show/hide channel list |
| **Left/Right** | Switch channel groups |
| **Up/Down** | Navigate channels / Select previous/next channel |
| **☰ MENU** | Open settings |
| **Back** | Hide channel list when visible |
| **D-Pad Center** | Show/hide channel list |
