# Entry Points

## Main Activity
- **File**: `MainActivity.kt`
- **Purpose**: Launch point for the application
- **Responsibilities**:
  - Playlist and EPG downloading
  - UI fragment management
  - Remote control input handling
  - Settings management

## Settings Activity
- **File**: `SettingsActivity.kt`
- **Purpose**: Configuration interface
- **Access**: Via ☰ MENU button

## Default Configuration
- **Default Playlist URL**: `http://openwrt.lan/iptv.m3u` (can be changed in settings)
